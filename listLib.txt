# cara menjalankan virtualenv di linux.

1. Install terlebih dahulu python/python3 di laptop
2. Lalu install virtualenv di laptop
3. Buat folder, lalu buat virtualenv di dalamnya dengan mengetikan perintah: virtualenv myenv
4. Aktifkan virtualenv dengan perintah: source myenv/bin/activate 
5. Install Library yang dibutuhkan seperti list di atas
6. Kalau sudah selesai, bisa dimatikan dengan perintah: deactivate
